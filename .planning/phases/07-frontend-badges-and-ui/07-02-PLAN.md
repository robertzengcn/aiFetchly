    The component receives an array of `FileOperationRecord` objects and renders them as a horizontal chip row. Each record becomes a `v-chip` with:
    - Color based on operation type: success (green) for create, warning (yellow) for overwrite, info (blue) for edit, error (red) for failed
    - Icon based on type: mdi-plus for create, mdi-file-refresh-outline for overwrite, mdi-pencil-outline for edit, mdi-alert-circle-outline for failed
    - Append icon: mdi-check-circle (success) or mdi-alert-circle (failure)
    - File basename extracted from `record.filePath` using `filePath.split(/[\\/]/).pop()`
    - Error message displayed below the chip when `record.success === false` and `record.error` exists
    - Click handler that invokes `AI_FILE_OPEN` IPC with `{ filePath: record.filePath }`
    - Cursor pointer and hover state on the entire chip (D-10, OPEN-03)

    For edit operations with a `diff` field:
    - A collapsible section toggled by clicking a small "diff" indicator or expanding the chip
    - Diff lines rendered in a `<pre>` block with CSS classes:
      - `.diff-add` for lines starting with `+` (green text, green background)
      - `.diff-remove` for lines starting with `-` (red text, red background)
      - `.diff-context` for lines starting with ` ` (gray text)
    - Limit diff display to first 50 lines with a "Show full diff" toggle for longer diffs
    - DO NOT use v-html for diff rendering -- use Vue text interpolation `{{ line }}` to prevent XSS

    Component props:
    ```typescript
    interface Props {
      records: readonly FileOperationRecord[];
    }
    ```

    Component emits: none (handles open-file internally via windowInvoke).

    Internal state:
    - `expandedDiffs: Set<string>` -- tracks which record IDs have expanded diff sections
    - `showFullDiff: Set<string>` -- tracks which records show full diff vs truncated

    The chip row uses `display: flex; flex-wrap: wrap; gap: 4px;` layout (D-02).

    Style guide:
    - Use Vuetify v-chip with `size="small"`, `variant="tonal"`, `density="compact"`
    - Diff colors: `.diff-add { color: #22863a; background-color: #f0fff4; }`, `.diff-remove { color: #cb2431; background-color: #ffeef0; }`, `.diff-context { color: #6a737d; }`

       const isTargetAssistantMessage = (messageIndex: number): boolean => {
         const messages = visibleMessages.value;
         // Walk backwards from the end to find the last assistant message
         for (let i = messages.length - 1; i >= 0; i--) {
           if (messages[i].role === 'assistant' && messages[i].messageType !== MessageType.TOOL_CALL && messages[i].messageType !== MessageType.TOOL_RESULT) {
             return i === messageIndex;
           }
         }
         return false;
       };
