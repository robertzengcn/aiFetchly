## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer -> IPC handler | Untrusted input from web page (query, location strings, numeric params) |
| IPC handler -> YandexMapsModule | Validated input passed to orchestration layer |
| IPC handler -> renderer (webContents.send) | Progress and result push to potentially destroyed window |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-02 | Denial of Service | yandexMaps-ipc.ts START handler | mitigate | Enforce MAX_CONCURRENT_SEARCHES = 3 limit |
