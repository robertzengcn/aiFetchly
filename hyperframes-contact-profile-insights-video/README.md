# Contact Profile Insights Tutorial Video

HyperFrames project for an English AiFetchly tutorial video explaining why and how to use Contact Profile Insights.

## Commands

```bash
npm run tts
npm run check
npm run render
```

## Output

- Voiceover: `media/contact-profile-insights-voiceover.wav`
- Rendered video: `renders/contact-profile-insights-tutorial.mp4`

## Notes

The composition uses app-like mockups rather than submitting real tasks in the live AiFetchly application. All visible contact data is sample data for demonstration.
